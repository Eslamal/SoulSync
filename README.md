## 📖 About The Project

**SoulSync** is a feature-rich, real-time Android application designed specifically for couples. It serves as a secure digital sanctuary where partners can communicate, play live synchronized games, make video calls, and document their shared life journey in a private timeline.

Built with a strong focus on clean architecture, smooth UI/UX, and real-time data synchronization, SoulSync demonstrates advanced Android development practices.

## ✨ Key Features

* 💬 **Real-time Rich Chat:** Send text, audio recordings (with duration UI), images, videos, and files seamlessly. Includes live "Online" and "Typing..." indicators.
* 🎮 **Live Synchronized Games:** Play together in real-time. Features *Truth or Dare*, *Who is More Likely*, *Spin the Wheel*, and a *Couple Quiz*. Game actions and spins are synced instantly using Firestore.
* 📅 **Shared Memories Timeline:** A dedicated space to save special dates, locations, and photo albums (using Image Sliders). Works with local offline caching (Room) and cloud backup.
* 📞 **Voice & Video Calls:** High-fidelity, real-time video and audio calling powered by ZegoCloud integration.
* 🔒 **Ultimate Privacy (App Lock):** Secure the app using a custom 4-digit PIN or device Biometric authentication (Fingerprint).
* 🌓 **Dynamic UI:** Full support for system-level Dark/Light modes and multi-language support (English/Arabic) with complete RTL/LTR layout adjustments.

## 🛠️ Tech Stack & Architecture

This project is built using modern Android development tools and follows the **MVVM (Model-View-ViewModel)** architectural pattern combined with the **Repository Pattern** for a clean separation of concerns.

* **Language:** Kotlin
* **UI:** XML with ViewBinding
* **Architecture:** MVVM, Clean Architecture concepts, Repository Pattern
* **Asynchronous Programming:** Coroutines & Kotlin Flow
* **Local Database:** Room SQLite
* **Remote Backend:** Firebase (Authentication, Firestore, Storage)
* **Media & Video:** Glide (Image Loading), Media3/ExoPlayer (Video Playback)
* **Communication:** ZegoCloud (Video/Voice Calling SDK)
